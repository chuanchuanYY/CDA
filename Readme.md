# 如何编译运行代码
~~~
需要先解除main.c 中的测试函数的注释才会有输出
~~~



1.在windows下
如果使用的是 gcc 编译器,直接运行run.ps1脚本即可。

~~~
./run.ps1
~~~



2.在Linux下也可以直接使用run.ps1中写的命令

~~~bash
gcc -I ./include -I ./Test -o main.exe ./main.c ./src/*.c ./Test/*.c
./main.exe
~~~



3.使用VsCode 需要设置task.json 编译器的路径

~~~
{
    "tasks": [
        {
            "type": "cppbuild",
            "label": "C/C++: gcc.exe build active file",
            // 修改command 的 编译器路径为你的编译器所在路径
            "command": "your compiler path",
            "args": [
                // "-fdiagnostics-color=always",
                // "-g",
                // "${file}",
                "-I",
                "./include",
                "-I",
                "./Test",
                "-o",
                "${fileDirname}\\${fileBasenameNoExtension}.exe",
                "./main.c",
                "./src/*.c",
                "./Test/*.c"
            ],
            "options": {
                "cwd": "${fileDirname}"
            },
            "problemMatcher": [
                "$gcc"
            ],
            "group": {
                "kind": "build",
                "isDefault": true
            },
            "detail": "Task generated by Debugger."
        }
    ],
    "version": "2.0.0"
}
~~~
